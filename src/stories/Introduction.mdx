# Welcome to Our Design System

This Storybook contains a collection of reusable UI components built with React and TypeScript. Our design system helps maintain consistency across applications while providing developers with well-documented, accessible components.

## What's Included

- **Input Components**: Form controls with validation and accessibility features
- **Feedback Components**: Toast notifications and alerts
- **Navigation Components**: Sidebar menus and navigation elements
- **Design Tokens**: Colors, typography, spacing, and other design foundations

## Who Maintains This

This design system is maintained by our frontend development team. For questions, suggestions, or bug reports, please:

- Create an issue in our repository
- Reach out to the design system team
- Join our Slack channel #design-system

## Getting Help

Each component includes:

- **Overview**: What the component does and when to use it
- **Playground**: Interactive controls to test all props
- **Examples**: Real-world usage scenarios
- **Documentation**: Props, accessibility guidelines, and best practices

## Quick Navigation

Use the sidebar to browse components by category:

- **Atoms**: Basic building blocks (Input, Button, etc.)
- **Molecules**: Simple component combinations
- **Organisms**: Complex UI sections
